# NodeStaticServer

## Install

Copy index.js file inside you static files, respecting this following architecture
```
<pre>
	- /img
	- /js
	- /css
	- index.html
	- otherPage.html
	- 404.html
	- index.js
</pre>
```
## Enchancement

 - [ ] Custom File System
 - [ ] Url Flexibility (use something else than file.html in the url)
