# NodeStaticServer

## Install

Copy index.js file inside you static files, respecting this following architecture
```
	- /img
	- /js
	- /css
	- index.html
	- otherPage.html
	- 404.html
	- index.js
```
Launch `node index.js` to start server and then browser `localhost:8080`
